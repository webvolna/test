# Telegraph Page Summarizer (Chrome Extension)

Расширение Chrome, которое по кнопке:
1. Извлекает текст текущей страницы.
2. Формирует краткий summary.
3. Создаёт страницу в `telegra.ph` с summary и ссылкой на исходную страницу.

## Установка

1. Откройте `chrome://extensions`.
2. Включите **Developer mode**.
3. Нажмите **Load unpacked** и выберите папку проекта.

## Использование

1. Откройте страницу, которую хотите суммаризировать.
2. Нажмите на иконку расширения.
3. Укажите `short name` (можно оставить значение по умолчанию).
4. Нажмите **Создать страницу**.
5. Получите ссылку на новую страницу в `telegra.ph`.

## Технические детали

- Манифест: Manifest V3.
- Логика публикации выполняется в `background.js`.
- Извлечение текста страницы выполняется через `chrome.scripting.executeScript`.
- Токен Telegraph создаётся автоматически через `createAccount` и сохраняется в `chrome.storage.local`.
